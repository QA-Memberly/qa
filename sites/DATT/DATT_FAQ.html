DATT App - Mobile Application FAQs

The following features are present in DATT App mobile application.

  • Profile
  • Add User
  • User List
  • Recent Activity
  • Smart Notes
  • Events
  • Diabetes Risk Score
  • Committee
  • About us
  • Support
  • FAQ’s
     

 

Click on “Forgot Username” hyperlink from the sign-in screen, provide the registered email address. The username will be sent to the registered email id.

 

 

Click on “Forgot Password” hyperlink from the sign-in screen, provide the registered email address. The password will be sent to the registered email id.

 

 

The password can be changed by clicking on Settings -> Reset Password.  The old password should be entered for confirmation. The new password has to be entered twice for submission. 

 

 

By clicking the “Add Person”, user information can be provided. The user will be receiving the credentials through email and SMS.

 

 

History of the daily symptoms will be displayed.

 

 

The activities of the users i.e. the message notifications, payment transactions, events, etc., will be displayed along with the status.

 

 

The advertisements of the sponsors will be displayed in the DATT application. If clicked on the advertisement, it will be navigated to the corresponding website.

 

 

The user can select membership from the profile screen. The membership can either be applied new or renewed. The lifetime membership is also available.

 

 

The Membership can either be applied by the user or through the supervisor. The membership details will be validated and approved by the supervisor.
